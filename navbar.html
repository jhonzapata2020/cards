En este proyecto estamos desarrollando un menu de navegacion con lista desplegable tipo categorias mediante listas desordendas de un div class navbar-fixed adentro una etiqueta nav qeu contiene primero al logo, luego un href con un data-target llamado menu-responsive en este caso con la clase sidenav-triguer de materialize. De igual manera en la parte de abajo estamos ubicando dos componentes cada uno en 6 columnas como ya sabemos el grid usa 12 columnas en toda la pantalla
Para ubicar este texto a la derecha de esta imagen hemos tomado 6 columnas con la clase col s6 luego en la etiqueta blockquote hemos agregado una etiqueta style con text-align: justify y asi hemos logrado justificar el texto. Si das click sobre la imagen podras notar que la imagen se agranda mostrandose de una mejor manera esto se logra con la clase materialboxed dentro de la etiqueta img seguido de la clase responsive-image para que la imagen repete las 6 columnas que se le asignaron y asi poder utilizar las otras 6 columnas restantes para este parrafo que se podria usar como parrafo descriptivo en un sitio web.
