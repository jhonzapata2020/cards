esta es la sintaxys "flow-text z-depth-5"Soy Flow Text Un defecto común que hemos visto en muchos FRAME-WORKS es la falta de soporte para texto verdaderamente receptivo. Mientras que los elementos de la página cambian de tamaño de forma fluida, el texto aún cambia de tamaño de forma fija. Para mejorar este problema, para páginas con mucho texto, hemos creado una clase que escala de forma fluida el tamaño del texto y la altura de línea para optimizar la legibilidad para el usuario. La longitud de la línea se mantiene entre 45 y 80 caracteres y la altura de la línea escala para ser más grande en pantallas más pequeñas. Para ver Flow Text en acción, cambie lentamente el tamaño de su navegador y observe cómo cambia el tamaño de este cuerpo de texto. ¡Use el botón de arriba para activar/desactivar el flujo de texto para ver la diferencia!
